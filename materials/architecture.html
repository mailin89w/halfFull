Safety
🛡️
Safety Layer
Every LLM output passes three independent checks before reaching the user.
Hard rules (code)
No diagnosis statements · "May suggest / could indicate" framing enforced · Immutable fields never overwritten · Disclaimer on every report
Schema validation
JSON schema check on all LLM output · Auto-repair for truncated responses · Falls back to offline mode on failure · Raw health data never logged
Tone rewrite
Secondary Groq pass (Llama 3.1-8B) softens alarming phrasing · Sensitive keys stripped from logs · Consent audit trail in Supabase
